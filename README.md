API Basejump:  Request Header Parser Microservice
===

Build a full stack JavaScript app 
that is functionally similar to this: 
[reference case](https://cryptic-ridge-9197.herokuapp.com/api/whoami/) and 
deploy it to Heroku.

  language and 
  operating system
